# Resume-Category-Predictor-Deploy-on-Streamlit
Developed a Resume Category Predictor using Bag of Words (BoW), TF-IDF, a Scikit-learn Pipeline, and Random Forest Classifier. The model takes resume text as input, processes it using NLP techniques, and accurately predicts the corresponding resume category. The application is deployed with Streamlit for an interactive user experience.
# 📄 Resume Category Predictor
📌 Overview

Resume Category Predictor is a Machine Learning and NLP-based application that automatically predicts the category of a resume from its text content. The project uses Natural Language Processing (NLP) techniques to clean and transform resume text before classifying it using a Random Forest model.

The application is built with Scikit-learn and deployed using Streamlit, providing a simple and interactive interface for users to upload or paste resume text and instantly receive the predicted category.

## 🚀 Features
1. Predicts resume category from text
2. NLP-based text preprocessing
3. Bag of Words (BoW) feature extraction
4. TF-IDF vectorization
5. Scikit-learn Pipeline implementation
6. Random Forest Classifier
7. Interactive Streamlit web application
8. Fast and accurate predictions
9. Easy-to-use interface

## 🛠️ Technologies Used
1. Python
2. Pandas
3. NumPy
4. Scikit-learn
5. Regular Expressions (Regex)
6. Streamlit

## 🧠 Machine Learning Workflow

6. TF-IDF Transformation

8. Train Random Forest Classifier

## 🎯 How It Works
1. Launch the Streamlit application.
2. Paste the resume text into the input box.
3. Click the Predict Category button.
4. The trained model analyzes the text and predicts the appropriate resume category.
